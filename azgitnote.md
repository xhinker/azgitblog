# My Git note  

1. Clone

>git clone <url>

2. Git init

>git init

3. add change
>git add *

3. commit
>git commit -m "message"

4. add remote server 
>git remote add origin <url>

5. pull from server
>git pull origin master

6. push to server
>git push -u origin master

7. save password
>git config --global credential.helper 'cache --timeout=3600'
